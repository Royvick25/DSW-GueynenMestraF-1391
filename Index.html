Selectores CSS

Los selectores CSS son patrones que se utilizan para seleccionar elementos HTML específicos y aplicarles estilos de diseño visual.
Los selectores CSS permiten al diseñador web aplicar estilos específicos a los elementos HTML y sus descendientes (hijos) basados en sus etiquetas HTML, atributos, clases, identificadores o la relación entre ellos.

Los selectores CSS son una herramienta poderosa para crear sitios web visualmente atractivos y bien organizados.

  • Selector de etiqueta: Selecciona todos los elementos de una etiqueta específica en una página web. Por ejemplo, el selector p seleccionaría todos los párrafos en la página.

  • Selector de clase: Selecciona todos los elementos que tienen un valor de clase específico.
    Por ejemplo, el selector .mi-clase seleccionaría todos los elementos que tienen una clase llamada "mi-clase".

  • Selector de ID: Selecciona el elemento que tiene un valor de identificación específico.
    Por ejemplo, el selector #mi-id seleccionaría el elemento que tiene un atributo "id" con el valor "mi-id".

  • Selector de atributo: Selecciona todos los elementos que tienen un atributo específico.
    Por ejemplo, el selector [type="text"] seleccionaría todos los elementos que tienen un atributo "type" con el valor "text".

  • Selector descendiente: Selecciona un elemento que es descendiente de otro elemento.
    Por ejemplo, el selector div p seleccionaría todos los elementos p que son descendientes de elementos div.
